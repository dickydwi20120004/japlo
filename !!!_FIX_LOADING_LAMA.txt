================================================
   FIX LOADING LAMA - LOCALHOST
================================================

MASALAH: 
Browser loading terus saat buka localhost:8000

PENYEBAB:
- MySQL masih recovery (lambat)
- Database connection timeout
- innodb_force_recovery masih aktif

================================================

SOLUSI 1: DISABLE FORCE RECOVERY (WAJIB!)
==========================================

MySQL sudah berhasil start, sekarang disable recovery:

STEP 1: Stop MySQL
------------------
1. Buka XAMPP Control Panel
2. Klik "Stop" di MySQL

STEP 2: Edit my.ini
-------------------
1. Di XAMPP, klik "Config" (MySQL)
2. Pilih "my.ini"
3. Cari baris:
   innodb_force_recovery = 1
   
4. HAPUS atau COMMENT (tambah # di depan):
   # innodb_force_recovery = 1
   
5. SAVE (Ctrl + S)
6. Tutup Notepad

STEP 3: Start MySQL Lagi
-------------------------
1. Kembali ke XAMPP
2. Klik "Start" di MySQL
3. Tunggu HIJAU

STEP 4: Restart Laravel Server
-------------------------------
1. Tutup CMD server (tekan Ctrl + C)
2. Jalankan lagi: 1_JALANKAN_INI.bat

STEP 5: Test Browser
--------------------
1. Close semua tab browser
2. Buka browser baru
3. Ketik: localhost:8000
4. Seharusnya cepat sekarang!

================================================

SOLUSI 2: CLEAR BROWSER CACHE
==============================

1. Tekan: Ctrl + Shift + Delete
2. Pilih "Cached images and files"
3. Time range: "All time"
4. Clear data
5. Close browser
6. Buka lagi

================================================

SOLUSI 3: GUNAKAN 127.0.0.1
===========================

Kadang "localhost" lambat di DNS.
Coba pakai IP langsung:

http://127.0.0.1:8000

================================================

SOLUSI 4: CEK PORT 8000
=======================

Pastikan port 8000 tidak bentrok:

netstat -ano | findstr :8000

Kalau ada PID lain, kill process atau
ganti port:

php artisan serve --port=8001

Lalu buka: http://localhost:8001

================================================

SOLUSI 5: RESTART SEMUA
========================

1. Stop Laravel server (Ctrl + C di CMD)
2. Stop MySQL di XAMPP
3. Stop Apache di XAMPP
4. Tunggu 5 detik
5. Start Apache
6. Start MySQL
7. Jalankan: 1_JALANKAN_INI.bat
8. Buka browser: localhost:8000

================================================

SOLUSI 6: BYPASS DATABASE (TEMPORARY)
======================================

Kalau masalah di database connection,
temporary disable:

Edit: .env

Ubah:
DB_CONNECTION=mysql

Jadi:
# DB_CONNECTION=mysql
DB_CONNECTION=

Restart server.

TAPI: Anda tidak bisa login tanpa database!
Ini hanya untuk test halaman home.

================================================

CEK APAKAH LOADING DARI MYSQL:
===============================

Buka CMD baru:

cd C:\xampp\htdocs\Japlo App
php artisan tinker

Ketik:
DB::connection()->getPdo();

Kalau LAMA atau HANG:
→ MySQL bermasalah
→ Ikuti SOLUSI 1

Kalau CEPAT (keluar object PDO):
→ MySQL OK
→ Ikuti SOLUSI 2-4
