================================================
   TROUBLESHOOTING
================================================

1. "localhost refused to connect"
   → Server belum jalan
   → Jalankan: 1_JALANKAN_INI.bat
   → Jangan tutup CMD-nya

2. "SQLSTATE connection refused"
   → MySQL belum running
   → Buka XAMPP, klik Start MySQL
   → Tunggu hijau

3. "Email atau password salah"
   → User belum ada di database
   → Jalankan: FIX_LOGIN_SEKARANG.bat
   → Atau register manual

4. Halaman loading lama
   → Clear cache browser (Ctrl + Shift + Delete)
   → Hard refresh (Ctrl + F5)
   → Restart server
