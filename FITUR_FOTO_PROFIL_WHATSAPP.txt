🐛 TROUBLESHOOTING:

ERROR: "The link [public/storage] already exists"
SOLUSI:
   └─ Storage link sudah dibuat, skip saja
   └─ Atau: hapus public/storage lalu run lagi

ERROR: Foto tidak muncul setelah upload
SOLUSI:
   └─ Cek: php artisan storage:link sudah dijalankan?
   └─ Cek: folder storage/app/public/profile_photos ada?
   └─ Cek: public/storage link ke storage/app/public?

ERROR: "413 Request Entity Too Large"
SOLUSI:
   └─ Foto terlalu besar (>2MB)
   └─ Compress foto dulu

ERROR: "Unauthenticated"
SOLUSI:
   └─ Login dulu sebelum upload foto
