🛡️ METODE VALIDASI YANG DIGUNAKAN:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

1. HTML5 Pattern
   pattern="[0-9]*"
   └─ Membatasi input hanya angka di HTML level

2. Input Mode Numeric
   inputmode="numeric"
   └─ Memunculkan keyboard angka di mobile

3. JavaScript onkeypress
   onkeypress="return event.charCode >= 48 && event.charCode <= 57"
   └─ Mencegah pengetikan huruf secara realtime
   └─ Hanya izinkan karakter angka 0-9

4. JavaScript oninput
   oninput="this.value = this.value.replace(/[^0-9]/g, '')"
   └─ Auto hapus semua karakter non-angka
   └─ Bekerja juga jika user paste text
