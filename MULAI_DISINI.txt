📌 LANGKAH MUDAH - IKUTI 3 STEP INI:


┌──────────────────────────────────────────────────────────────────────────────┐
│  STEP 1: Start XAMPP MySQL                                                   │
└──────────────────────────────────────────────────────────────────────────────┘

    1. Buka XAMPP Control Panel
    2. Klik tombol [Start] pada MySQL
    3. Tunggu sampai statusnya HIJAU/Running


┌──────────────────────────────────────────────────────────────────────────────┐
│  STEP 2: Buat User Demo (Hanya Sekali)                                       │
└──────────────────────────────────────────────────────────────────────────────┘

    Double-klik file:  BUAT_USER_DEMO.bat
    
    User yang akan dibuat:
    - Email: demo@japlo.com
    - Password: password123


┌──────────────────────────────────────────────────────────────────────────────┐
│  STEP 3: Start Server Laravel                                                │
└──────────────────────────────────────────────────────────────────────────────┘

    Double-klik file:  START_SERVER.bat
    
    ⚠️ JANGAN TUTUP WINDOW yang muncul!
    Server harus tetap running.

🆘 TROUBLESHOOTING


    ❌ Server tidak jalan?
       → Pastikan XAMPP MySQL sudah running
       → Jalankan ulang START_SERVER.bat
    
    ❌ Login gagal?
       → Pastikan email: demo@japlo.com
       → Pastikan password: password123
       → Atau jalankan ulang BUAT_USER_DEMO.bat
    
    ❌ Halaman tidak muncul?
       → Cek di browser: http://localhost:8000
       → Pastikan server Laravel running (window START_SERVER.bat jangan ditutup)
