🌐 Cara Buka Japlo App di Web

Panduan Lengkap Step-by-Step

⚠️ MASALAH SAAT INI:

Server Laravel TIDAK RUNNING

Port 8000 closed - Aplikasi belum bisa diakses di browser

✅ Solusi Paling Mudah

1

Buka File Explorer

Pergi ke folder: c:\xampp\htdocs\Japlo App\

2

Cari dan Double-Click File

Nama file: START_SERVER.bat

⚠️ Window Command Prompt akan terbuka - JANGAN DITUTUP!

3

Tunggu Server Running

Akan muncul pesan:

INFO Server running on [http://127.0.0.1:8000]
4

Buka Browser

Chrome, Firefox, atau Edge

Ketik di address bar:

http://localhost:8000
🚀 BUKA JAPLO APP
5

Login

Email: demo@japlo.com

Password: password123

🎉 Berhasil!

Setelah login, Anda akan melihat Dashboard dengan 8 icon menu layanan:

🏍️ OJEK/TAXI | 🍽️ KULINER | 📢 PROMOSI | 🏥 KESEHATAN
🛍️ PRODUK | 🖨️ PENCETAKAN | 🔥 TRENDING | 👥 SOSIAL

📝 Alternatif: Via Command Prompt

Jika cara di atas tidak berhasil:

1

Buka CMD

Tekan Windows + R, ketik cmd, Enter

2

Masuk ke Folder Japlo

cd "c:\xampp\htdocs\Japlo App"
3

Jalankan Server

php artisan serve

⚠️ JANGAN TUTUP WINDOW CMD!

4

Buka Browser

🚀 BUKA JAPLO APP

✅ Checklist Sukses

  • Command window menampilkan "Server running"
  • Window CMD TIDAK ditutup
  • Browser terbuka di http://localhost:8000
  • Halaman Japlo muncul (ada logo 🏍️)
  • Klik tombol "Masuk"
  • Login dengan demo@japlo.com / password123
  • Dashboard muncul dengan 8 icon menu
  • 🎉 SUKSES!

🚨 Troubleshooting

❌ "This site can't be reached"

Solusi: Server belum running. Ulangi langkah 1-3

❌ Halaman blank/putih

Solusi: Tekan F5 atau Ctrl+Shift+R untuk refresh

❌ HTTP 500 Error

Solusi: Ada error di aplikasi. Jalankan:

php artisan cache:clear

❌ Port already in use

Solusi: Hentikan proses PHP:

taskkill /F /IM php.exe
