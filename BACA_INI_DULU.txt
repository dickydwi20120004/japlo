  php artisan view:clear
  php artisan config:clear
  php artisan route:clear
  Restart browser

Problem: Layout tidak responsive?
Solution: Hard refresh (Ctrl+Shift+R)

Problem: Server tidak running?
Solution:
  taskkill /F /IM php.exe
  php artisan serve
