LANGKAH UNTUK MELIHAT PERUBAHAN:

1. Pastikan server Laravel sudah berjalan:
   - Jalankan file: 1_JALANKAN_INI.bat
   
2. Buka browser dan hapus cache browser:
   - Chrome/Edge: Tekan Ctrl + Shift + Delete
   - Pilih "Cached images and files"
   - Klik "Clear data"
   
3. Atau gunakan mode Incognito/Private:
   - Chrome/Edge: Ctrl + Shift + N
   - Firefox: Ctrl + Shift + P
   
4. Buka halaman:
   - Register: http://localhost:8000/register
   - Login: http://localhost:8000/login

5. Jika masih belum berubah, coba:
   - Hard refresh: Ctrl + Shift + R
   - Atau Ctrl + F5
