## 🔧 TROUBLESHOOTING

### Problem: Server tidak running
**Solusi:**
```bash
# Kill semua proses PHP
taskkill /F /IM php.exe

# Start ulang
php artisan serve
```

### Problem: Error "Route [customer.dashboard] not defined"
**Status:** ✅ SUDAH DIPERBAIKI!
- Semua route sudah diganti ke `route('dashboard')`

### Problem: Page not found / 404
**Solusi:**
```bash
php artisan route:clear
php artisan config:clear
php artisan cache:clear
```

### Problem: Login tidak bisa
**Solusi:**
- Clear browser cookies atau gunakan incognito mode
- Pastikan email: demo@japlo.com
- Pastikan password: password123

### Problem: CSS tidak load
**Solusi:**
- Hard refresh: Ctrl + Shift + R
- Clear browser cache
- Check console (F12) untuk errors
