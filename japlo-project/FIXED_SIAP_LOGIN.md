## 🔧 TROUBLESHOOTING:

### Jika Halaman Tidak Muncul:
1. Pastikan server running (cek terminal)
2. Clear browser cache (Ctrl+Shift+Del)
3. Refresh halaman (F5 atau Ctrl+R)
4. Coba buka di incognito/private mode

### Jika Login Gagal:
1. Pastikan sudah register terlebih dahulu
2. Cek email & password benar
3. Pastikan database running (XAMPP MySQL aktif)
