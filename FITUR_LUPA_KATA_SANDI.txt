🔄 ALUR LUPA PASSWORD:
━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━━

STEP 1: User Lupa Password
   └─ Buka halaman Login
   └─ Klik link "Lupa kata sandi?"

STEP 2: Masukkan Email
   └─ Halaman Forgot Password terbuka
   └─ Masukkan email yang terdaftar
   └─ Klik "Kirim Link Reset"

STEP 3: Verifikasi Email
   └─ Sistem cek email di database
   └─ Jika tidak ada → Error
   └─ Jika ada → Generate token

STEP 4: Generate Token
   └─ Token random 64 karakter
   └─ Simpan di session (demo)
   └─ Set expiry 60 menit

STEP 5: Kirim Link (Demo Mode)
   └─ Dalam real app: Kirim email
   └─ Demo mode: Langsung redirect ke halaman reset
   └─ URL: /reset-password/{token}?email={email}

STEP 6: Halaman Reset Password
   └─ Validasi token & email
   └─ Cek expiry time
   └─ Show form password baru

STEP 7: Update Password
   └─ User input password baru
   └─ Validasi: min 8 karakter, konfirmasi cocok
   └─ Hash password baru (bcrypt)
   └─ Update database
   └─ Clear session token

STEP 8: Redirect ke Login
   └─ Success message
   └─ User login dengan password baru
