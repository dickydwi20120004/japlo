📝 EDIT PROFIL:

STEP 1: Buka Halaman Profil
   └─ Klik nama user di navbar → "Profil"

STEP 2: Klik Tombol "Edit Profil"
   └─ Modal akan terbuka

STEP 3: Edit Data
   └─ Ubah nama, email, telepon, atau alamat
   └─ Nomor telepon hanya bisa angka

STEP 4: Klik "Simpan Perubahan"
   └─ Data tersimpan ke database
   └─ Halaman reload dengan data baru
   └─ Muncul pesan sukses

🔑 GANTI PASSWORD:

STEP 1: Buka Halaman Profil
   └─ Klik nama user di navbar → "Profil"

STEP 2: Klik Tombol "Ganti Password"
   └─ Modal akan terbuka

STEP 3: Isi Form
   └─ Masukkan password lama
   └─ Masukkan password baru (min 8 karakter)
   └─ Konfirmasi password baru
   └─ Gunakan icon mata untuk show/hide

STEP 4: Klik "Ganti Password"
   └─ Validasi password lama
   └─ Validasi password baru & konfirmasi
   └─ Password tersimpan (encrypted)
   └─ Muncul pesan sukses
