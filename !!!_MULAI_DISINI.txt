TROUBLESHOOTING:
================

Q: "MySQL refused to connect"
A: MySQL belum running
   → Buka XAMPP, Start MySQL

Q: "localhost refused to connect"
A: Laravel server belum jalan
   → Klik 2x: 1_JALANKAN_INI.bat

Q: "Email atau password salah"
A: User belum dibuat
   → Klik 2x: SETUP_LENGKAP.bat

Q: Setup lambat atau hang?
A: Normal, MySQL baru recovery
   → Tunggu saja 2-3 menit
   → Atau restart XAMPP (Stop All > Start All)

Q: Browser loading lama?
A: Clear cache browser
   → Ctrl + Shift + Delete
   → Hard refresh: Ctrl + F5
