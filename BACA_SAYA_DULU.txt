❓ ALTERNATIF CARA (Tanpa Batch File):
═══════════════════════════════════════════════════════════════════════════════

   1. Tekan Windows + R
   2. Ketik: cmd
   3. Tekan Enter
   4. Ketik: cd "c:\xampp\htdocs\Japlo App"
   5. Tekan Enter
   6. Ketik: php artisan serve
   7. Tekan Enter
   8. Tunggu sampai muncul "Server running"
   9. JANGAN TUTUP WINDOW CMD!
   10. Buka browser: http://localhost:8000

🆘 TROUBLESHOOTING:
═══════════════════════════════════════════════════════════════════════════════

   ❌ File 1_JALANKAN_INI.bat tidak ada?
      → Refresh folder (F5)
      → Atau gunakan cara alternatif (tanpa batch file)

   ❌ Double-click tapi langsung tertutup?
      → Klik kanan → Run as Administrator

   ❌ Browser menunjukkan "can't be reached"?
      → Pastikan server sudah running
      → Lihat window CMD masih terbuka
      → Cari tulisan "Server running"

   ❌ Halaman blank/putih?
      → Tekan F5 untuk refresh
      → Atau Ctrl + Shift + R

   ❌ Masih tidak bisa?
      → Hubungi support atau cek dokumentasi lengkap
